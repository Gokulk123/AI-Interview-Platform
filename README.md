# 🚀 AI Interview Platform

An AI-powered Full Stack Interview Preparation Platform built with **React.js, Node.js, Express.js, PostgreSQL, and Groq AI**.

The platform helps users prepare for technical interviews by generating AI-based interview questions based on their experience level and job role. Users can also review their interview history and receive AI-powered feedback.

# 📖 Project Overview

AI Interview Platform is a production-ready full-stack web application designed to help software developers prepare for technical interviews.

Users can:

- Register and Login securely
- Create mock interviews
- Generate AI interview questions
- Review interview history
- Receive AI-generated interview questions using Groq LLM

This project demonstrates real-world full-stack development, AI integration, REST API development, authentication, cloud deployment, and PostgreSQL database management.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Gokulk123/AI-Interview-Platform.git
```

---

## Install Frontend

```bash
cd frontend

npm install

npm run dev
```

Runs on

```
http://localhost:5173
```

---

## Install Backend

```bash
cd backend

npm install

npm start
```

Runs on

```
http://localhost:5000
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```
DATABASE_URL=your_database_url

JWT_SECRET=your_secret_key

GROQ_API_KEY=your_groq_api_key

PORT=5000
```
